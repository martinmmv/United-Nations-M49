# United Nations M49 Analysis

This project explores the United Nations M49 standard country and area codes using Python.

The standard country or area codes for statistical use (M49) are prepared by the Statistics Division of the United Nations Secretariat primarily for use in its publications and databases. For more information, see: https://unstats.un.org/unsd/methodology/m49/

## Objective
The goal is to understand how the UN organizes countries and territories.

## Dataset
Source: United Nations Statistics Division (M49 standard)

## Repository structure
- `UN-M49.csv` → source dataset
- `United_Nations_M49.ipynb` → exploratory analysis notebook

## How to run
1. Clone the repository
2. Install dependencies
3. Open the notebook
